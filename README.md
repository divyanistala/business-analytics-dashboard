# Tableau Analytics Dashboard

##  Project Snapshot
- Objective: Visualise sales performance and customer KPIs through interactive dashboards  
- Tools: Tableau, Excel  
- Focus: Data storytelling, KPI tracking, and trend analysis  
- Outcome: Delivered clear, actionable insights to support business decision-making  

---

##  Overview
This project presents an interactive business analytics dashboard developed in Tableau to analyse sales performance and customer key performance indicators (KPIs). The dashboard is designed to communicate insights effectively through data storytelling and trend analysis, supported by Excel-based data preparation and modelling.

---

##  Dataset
- Superstore dataset used for sales and customer analysis  
- Available in: `data/superstore_dataset.xlsx`

---

##  Dashboard
The dashboard provides:
- Sales performance trends over time  
- Key customer metrics and segmentation  
- Comparative analysis across categories and regions  

 View dashboard:
- `dashboard/dashboard-overview.pdf`

---

##  Methodology
- Cleaned and structured data using Excel  
- Designed dashboard layout for clarity and usability  
- Built interactive visualisations in Tableau  
- Applied data storytelling principles to highlight key insights  

---

##  Key Insights
- Sales trends highlight periods of strong and weak performance, supporting better planning  
- Customer segmentation reveals high-value groups and purchasing patterns  
- Clear visualisation improves communication of business insights  
- Dashboard enables faster, data-driven decision-making
